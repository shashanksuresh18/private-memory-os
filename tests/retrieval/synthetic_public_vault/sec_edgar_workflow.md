# SEC EDGAR Public Filing Workflow

The SEC's EDGAR system accepts electronic filings from public companies.
Filers receive a CIK number and a filer-management account on first
submission.

## Common Filing Types

10-K, 10-Q, and 8-K cover annual, quarterly, and current report disclosures
respectively. Proxy statements file as DEF 14A. Insider trades file on Form
4 within two business days.

## Public Access

All accepted filings publish to EDGAR's full-text search within minutes of
acceptance. There is no fee for public access. Bulk downloads are available
via the EDGAR FTP and HTTPS endpoints.
